Pengertian Microsoft word

Microsoft Office Word atau Microsoft Word, merupakan sebuah perangkat lunak pengolah kata yang dikeluarkan oleh Microsoft.

Ini adalah sebuah aplikasi pengolah kata yang dapat membantu dalam menyelesaikan kegiatan apa pun yang berkaitan dengan dokumen, teks dan sejenisnya.

Dan sebagai contohnya saja Anda dapat membuat, mengedit, serta memformat sebuah dokumen. Hasil dari dokumen tersebut dapat disimpan dalam bentuk softcopy ataupun hardcopy. Dokumen ini biasanya dapat berupa berbagai hal, seperti surat, brosur, buku, kartu nama, jurnal dan lainnya.

Pada tahun 1983, Microsoft Office Word atau Microsoft Word pertama kali diterbitkan dengan nama Multi-Tool Word untuk Xenix.

Fungsi Microsoft Word

  1. Digunakan untuk keperluan kantor ataupun keperluan membuat dokumen-dokumen para pelajar.
  2. Digunakan untuk membuat surat
  3. Digunakan untuk membuat dokumen.
  4. Digunakan untuk membuat tabel dan juga grafik.

Cara membuka Microsoft Word

  1. Nyalakan perangkat yang ada di rumah anda Baik itu komputer atau laptop sehingga muncul ke halaman utama Windows.
  2. Klik menu start dengan logo Windows di pojok kiri bawah
  3. Cari Microsoft Word kemudian klik kiri sekali atau Klik Kanan untuk memilih option open
  4. Setelah masuk ke Microsoft Word, klik pilihan dokumen kosong untuk memulai

Mengenal fungsi icon yang paling umum di Microsoft Word

  1. Font size: digunakan untuk mengubah ukuran huruf.
  2. Font : untuk mengubah jenis dan bentuk huruf.
  3. Font Color : digunakan untuk memberi atau mengganti warna pada huruf.
  4. Italic : digunakan untuk mengubah huruf menjadi miring.
  5. Bold : digunakan untuk mengubah huruf menjadi tebal.
  6. Underline : digunakan untuk menambah garis bawah pada kalimat atau kata
  7. Highlight : digunakan untuk memberi warna latar pada huruf.
  8. Center : digunakan untuk membuat paragraf menjadi rata tengah.
  9. Align left : digunakan untuk membuat paragraf menjadi rata kiri.
  10. Align right : digunakan untuk membuat paragraf menjadi rata kanan.
  11. Justify : digunakan untuk membuat paragraf rata kanan kiri.
  12. Bullets : digunakan untuk membuat daftar perpoin.
  13. Line : spacing digunakan untuk mengatur jarak spasi.
  14. Decrease : indent digunakan untuk memajukan paragraf 1 Tab.
  15. Increase : indent berfungsi untuk memundurkan paragraf 1 Tab.
  16. Numbering : digunakan untuk membuat daftar berpoin angka.
  17. Borders : digunakan untuk menambah garis batas.
  18. Title bar : berfungsi sebagai pemberi informasi judul atau nama file yang sedang dikerjakan
  19. Ruler : berfungsi sebagai pengukur pada bagian apa tas dan samping kiri dokumen Word yang mana menjadi indikator lebar dan tingginya sebuah halaman, ini juga bisa digunakan untuk mengatur margin

Shortcut Keyboard Microsoft Word

  1. Ctrl + A : Memilih Teks
  2. Ctrl + B: Menebalkan Teks
  3. Ctrl + C: Menyalin teks
  4. Ctrl + D : Membuka Jendela Font Formatting
  5. Ctrl + E: Membuat Tulisan Jadi di Tengah
  6. Ctrl + F: Mencari Kata
  7. Ctrl + G: Ke Halaman Tertentu
  8. Ctrl + H: Mengganti kalimat dengan kalimat lain
  9. Ctrl + I: Membuat Tulisan miring (Italic)
  10. Ctrl + J: Membuat Tulisan Rata Kiri dan Kanan (Justify)
  11. Ctrl + K: Menambah Hyperlink
  12. Ctrl + L: Membuat Tulisan Rata Miring (Align Left)
  13. Ctrl + M: Membuat Paragraf di Kiri
  14. Ctrl + N: Membuat File Baru
  15. Ctrl + O: Membuka File
  16. Ctrl + P: Print
  17. Ctrl + Q: Menghapus Editing
  18. Ctrl + R: Membuat Tulisan Rata Kanan (Align Right)
  19. Ctrl + S: Menyimpan
  20. Ctrl + T: Membuat Paragraf Menggantung
  21. Ctrl + U: Menambah Garis Bawah (Underline)
  22. Ctrl + V: Menyalin Teks (Paste)
  23. Ctrl + W: Menutup Jendela Word
  24. Ctrl + X: Cut
  25. Ctrl + Y: Redo
  26. Ctrl + Z: Undo
